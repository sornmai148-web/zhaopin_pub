### 本地开发阶段运行以下指令

安装项目全用的依赖

```bash
npm install
# or
yarn install
```

```bash
npm run dev
# or
yarn dev
```

默认开发模式的开放端口 ：3000 ，本地域名 ： localhost,127.0.0.1

```bash
整个链接 ：http://localhost:3000
```

### 正式部署阶段运行以下几个指令

安装项目全用的依赖

```bash
npm install
# or
yarn install --check-files --frozen-lockfile
```

打包项目

```bash
npm build
# or
yarn build
```

生产运行

```bash
npm start
# or
yarn start
```

```bash
整个链接 ：http://localhost:3000
```
